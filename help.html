Help and Support

This game works on Windows only. If you are using Windows and don't know how to run or an error has occurred, please read:

1. Click the Download button in the Google Drive link

2. Open the "game.bat" file. You will need to extract the file.

3. If you are using Windows 7 or older, you can upgrade to Windows 8 or newer.

4. This version might have some bugs. We are trying to fix them.

NOTE: DO NOT DELETE SUBFILES OR SUBFOLDERS, THE GAME WON'T WORK!
